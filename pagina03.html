Elementos de un Sistema de Informacion Computacional
Equipo Computacional: hardware necesario para que el sistema de Informacion funcione,integrado por computadoras y equipo periferico conectado a ellas.
Recursos humanos: son las personas que interactuan con el sistema de Informacion,suministran datos o utilizan el sistema para generar datos.
Fuente:son los datos e Informacion que se introducen en el sistema, es decir,son las entradas requeridas para el sistema Informatico funcione.
Programas: software ejecutado por la computadora,el cual procesa la informacion de entrada para generar los resultados esperandos.
Telecomunicaciones:abarca el software y hardware que permite la transmision en forma electronica de texto e imagenes,entre otros.
Politicas y reglas de operacion: aplicadas a los procesamientos y mecanismos para hacer que una aplicacion de computadora trabaje de acuerdo con estas o regir la parte fundamental del proceso administrativo de la empresa.
